```java
public class MyClass {

    private String title;

    public String getTitle() {
        return title;
    }

    public void setTitle(String title) {
        this.title = title;
    }

    public MyClass() {
    }

}
```

```java
        myClass = new MyClass();
        myClass.setTitle("処理開始");
        String title = myClass.getTitle();
        System.out.println(title);
```
